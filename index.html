The Optimal Architecture Configuration

The Neural Architecture Search process identified an optimal 5-stage architecture containing 13 MBConv layers, distributed strategically for maximum efficiency and feature extraction.

Stage Channels Depth (Blocks) Stride Block Type & Kernel
1: Foundation 16 1 1 Single block with 3x5 kernel and expansion factor 3
2: Initial Expansion 24 2 2 Block 1: 3x5 kernel (exp 3), Block 2: 3x3 kernel (exp 3)
3: Deep Processing 32 4 2 All 4 blocks use uniform 6x3 kernels for consistent feature extraction
4: Mixed Receptive Fields 64 4 2 Heterogeneous design: 6x3, 3x5, 6x5, 3x3 kernels for diverse feature capture
5: Final Representation 96 2 1 Two 6x5 kernel blocks for high-level feature aggregation

Total Core Layers: 13 MBConv blocks (plus input/output conv and classifier).
